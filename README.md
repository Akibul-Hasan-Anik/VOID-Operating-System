# VOID Operating System

A basic custom operating system developed using the Cosmos framework and C# programming language.

## Overview

This project was created as a beginner-friendly demonstration of how a custom operating system can be developed using the Cosmos framework and Visual Studio. This operating system includes fundamental OS functionalities and demonstrates low-level system development concepts using C#. It serves as an introduction to operating system architecture, kernel development, and bootable OS creation.

## Features

* Custom operating system development
* Basic kernel implementation
* Bootable ISO generation
* Cosmos framework integration

## Technologies Used

* C#
* Cosmos OS Framework
* Visual Studio

## Project Structure

```text

```

* Operating system development
