# VOID Operating System

A basic custom operating system developed using the Cosmos framework and C# programming language.

## Overview

This project was created as a beginner-friendly demonstration of how a custom operating system can be developed using the Cosmos framework and Visual Studio.

The operating system includes fundamental OS functionalities and demonstrates low-level system development concepts using C#. It serves as an introduction to operating system architecture, kernel development, and bootable OS creation.

## Features

* Custom operating system development
* Basic kernel implementation
* Bootable ISO generation
* Cosmos framework integration
* Beginner-friendly operating system project
* Easy to customize and expand

## Technologies Used

* C#
* Cosmos OS Framework
* Visual Studio
* Operating System Development Concepts

## Project Structure

```text
VOID-Operating-System/

├── VOID() OS Report.docx      # Detailed project report and user guideline
└── README.md                  # Project documentation and setup instructions
```

## How to Run the Project

1. Clone the repository.
2. Open the project folder in Visual Studio.
3. Install the Cosmos framework and required dependencies.
4. Open the project source code and build the project.
5. Run the operating system using:
   * VMware
   * VirtualBox
   * QEMU
   * Cosmos Emulator
6. You can also use the provided ISO file for boot testing.
